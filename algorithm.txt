ACTIVITY 1: Add Two Numbers
Step 1: Start.
Step 2: Ask the user to enter the first number. Store it as num1.
Step 3: Ask the user to enter the second number. Store it as num2.
Step 4: Compute sum = num1 + num2.
Step 5: Display sum.
Step 6: End.

ACTIVITY 2: Compute the Average of Three Quiz Scores
Step 1: Start.
Step 2: Ask the user to enter the first grade. Store it as grade1.
Step 3: Ask the user to enter the second grade. Store it as grade2.
Step 4: Ask the user to enter the third grade. Store it as grade3.
Step 5: Compute average = (grade1 + grade2 + grade3) / 3.
Step 6: Display average.
Step 7: End.

ACTIVITY 3: Determine if a Number is Even or Odd
Step 1: Start.
Step 2: Ask the user to enter a number. Store it as number.
Step 3: Check if number MOD 2 equals 0.
Step 4: If yes, display "Even".
Step 5: If no, display "Odd".
Step 6: End.
